Esta seção descreve os endpoints da GoMoney API destinados à **adição de saldo em customers**. São três operações disponíveis: criar uma adição de saldo, listar todas as adições do usuário autenticado e consultar uma adição específica por ID.

**URL base:** `https://gmc-api.gomoney.me/api`

## Autenticação

Todas as requisições exigem os seguintes cabeçalhos:

| Cabeçalho       | Valor                                                                    |
| --------------- | ------------------------------------------------------------------------ |
| `Authorization` | `Bearer {{accessToken}}`                                                 |
| `secret-key`    | `{{secretKeyBase64}}`                                                    |
| `Origin`        | `{{allowedOrigin}}` — deve corresponder ao domínio aprovado pelo suporte |

<Warning>
  O playground desta documentação envia as requisições pelo servidor da Mintlify (não diretamente pelo navegador), então o cabeçalho `Origin` que você digitar é encaminhado normalmente. Ainda assim, ele precisa ser um domínio previamente aprovado pelo suporte GoMoney para a sua integração — caso contrário a API retornará `401`.
</Warning>
