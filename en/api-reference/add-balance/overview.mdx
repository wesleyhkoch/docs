This section describes the GoMoney API endpoints for **adding balance to customers**. Three operations are available: create a balance addition, list all additions for the authenticated user, and query a specific addition by ID.

**Base URL:** `https://gmc-api.gomoney.me/api`

## Authentication

All requests require the following headers:

| Header          | Value                                                                    |
| --------------- | ------------------------------------------------------------------------ |
| `Authorization` | `Bearer {{accessToken}}`                                                 |
| `secret-key`    | `{{secretKeyBase64}}`                                                    |
| `Origin`        | `{{allowedOrigin}}` — must match the domain approved by support         |

<Warning>
  This documentation's playground sends requests through the Mintlify server (not directly from the browser), so the `Origin` header you type in is forwarded as-is. It still needs to be a domain previously approved by GoMoney support for your integration — otherwise the API returns `401`.
</Warning>

## Available endpoints

<CardGroup cols={2}>
  <Card title="Add balance" href="/en/api-reference/add-balance/create" icon="circle-dollar-to-slot">

  </Card>
  <Card title="List additions" href="/en/api-reference/add-balance/list" icon="list">

  </Card>
